# Commandパターンとは
デザインパターンの一種でリクエストの呼び出し元と、実装処理を分離することができる。

# いつ使うのか
- 今後いろいろなコマンドの追加が予想される時
- コマンドの履歴を管理したい時
- コマンドをキューで管理したい時
- 再利用性を高めたい時
- 単一責任原則を適用したい時
